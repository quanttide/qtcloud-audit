# 工程标准

## 审计规则（Audit Rule）  

定义：用于评估某一业务领域运行状态是否健康、合规或高效的结构化判断标准。审计规则明确“从哪些维度、依据什么指标、采用何种阈值”来识别潜在风险或异常，是开展审计工作的逻辑基础。

## 审计规划（Audit Plan）  

定义：在特定审计规则指导下，对审计执行的时间安排、覆盖范围、频率（定期或不定期）及触发条件所作的系统性安排。审计规划决定“何时审、审谁、怎么启动”，确保审计活动有序、高效、有的放矢。

## 审计问题（Audit Issue）

定义：在执行审计过程中发现的具体问题、偏差或风险点，通常表现为与审计规则不符的事实或数据异常。审计事项是审计工作的最小产出单元，需包含问题描述、证据支撑、责任主体及整改建议，并支持闭环跟踪。
